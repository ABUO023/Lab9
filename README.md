# CSI 3540 - Lab 9 - AJAX & DOM Events

Complete AJAX web application with real-time user list and messaging.

# Demo :


## Quick Start

```bash
docker-compose up --build
```

Then open: **http://localhost:8080/lab9/login.html**

## Features

- **Login/Logout** - User authentication
- **Real-time Users List** - Auto-updates every 2 seconds
- **Dynamic Dropdown** - AJAX-fetched values
- **Broadcast Messaging** - Real-time message delivery
- **Thread-safe** - Synchronized concurrent access

## Stop

```bash
docker-compose down
```
